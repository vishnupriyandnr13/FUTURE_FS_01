# Portfolio Website – Vishnu Priya

A responsive,full-stack portfolio showcasing my projects, resume, and a working contact form powered by Node.js and Express.

---

##  Key Features

- Responsive layout using Bootstrap 4
- Contact form that sends emails via Nodemailer
- Resume download (PDF)
- Project showcase section
- Social media integration (LinkedIn, GitHub, WhatsApp)

---

## Tech Stack

- **Frontend**: HTML, CSS, JavaScript, Bootstrap, Font Awesome  
- **Backend**: Node.js, Express, Nodemailer, CORS

### Frontend:

1. Open `frontend/index.html` in a browser, or deploy via GitHub Pages / Netlify.

### Backend

```bash
cd backend
npm install


Create a .env file with your email credentials:

EMAIL_USER=your_email@gmail.com
EMAIL_PASS=your_app_password


Run the server:
npm start

Usage:

Fill out the contact form; your message is sent to the configured email account.

Download the PDF resume via the link provided.

View project cards with descriptions and images in the Projects section.
